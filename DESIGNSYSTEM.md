# Dusk&Co Design System

## Typography

### Font Families
1. **Roboto Slab** (Primary) — Google Fonts
   - Bold: Headlines (H1)
   - Regular: Body copy, paragraphs (H4, body)
   - Light: Subheadings (H2), fine print, all-caps editorial
   - Thin: Available for extra fine print (use sparingly)

2. **Expansiva** (Secondary) — Local font file
   - Regular only: Special headings, always bold, sizes 14–72pt

### Type Scale (Web Adapted)
| Level | Font | Weight | Size (desktop) | Size (mobile) | Tracking |
|-------|------|--------|----------------|---------------|----------|
| H1 | Roboto Slab | Bold | 48px (3rem) | 32px (2rem) | 0 |
| H2 | Roboto Slab | Light | 21px (1.3rem) | 18px (1.125rem) | 0 |
| H3 | Expansiva | Regular | 14–72px | Scale down | 0 |
| H4 | Roboto Slab | Bold | 10–14px | 10–14px | 0 |
| Body | Roboto Slab | Regular | 12–16px | 14–16px | 0.015em |
| Caption | Roboto Slab | Light | 8px | 10px | 0.1em (all caps) |
| Fine print | Roboto Slab | Light | 7.5px | 10px | 0 |

### Rules
- Headlines: dark and bold
- Subheadings: light-weight with soft grey tones
- On dark backgrounds: increase line-height for readability
- Expansiva is ALWAYS bold and used for special display headings

## Visual Identity

### Logo Usage
- Primary logo: wide horizontal — official documents, website header, packaging
- Responsive logo (logomark): small spaces, profile pics, favicons, mobile headers
- Alternate logo: narrow vertical spaces, tags
- Spacing: 1x padding on all sides (1x = height of "D" in logo)

### Graphic Elements
- Rectangles and tags — NEVER circles
- Pattern motif (brick-like, stackable, scalable)
- Graffiti-like accent scribbles for emphasis on text/images
- Portrait orientation preferred for images; landscape acceptable where needed
